  • Step 1: Finding options.ini

    This one's simple.

    Navigate to C:/Users/[your username]/Documents/My Games/ Binding of Isaac Repentance/

    (If you can't find this file, try checking another drive)

    Now to change the setting.

    Open the options.ini file in any text editor, and find the "EnableDebugConsole" line.
    If it is set to 0, replace it with 1.

  • Congrats! You've enabled the Debug Console

    For a list of commands you can use, https://wofsauge.github.io/IsaacDocs/rep/tutorials/DebugConsole.html?h=debug#commands
