Setting up your mod

  • Step 1: Creating a mod folder

    First you have to navigate to the mods folder.

    In repentance, you can do this by going to the game's page on steam, clicking the "Manage" button, going to the "Manage" tab, and choosing "Browse local files". Then, open or create the "mods" folder.

    Now to create the mod folder.

    Inside the mods folder, create a folder with any name. Inside of this folder, create 2 folders, called "content" and "resources", and a text file called "main.lua", (we won't be using this yet) Mods also require a "metadata.xml", but the game will create this automatically.

  • Step 2: Lets check that it works (optional)

    Lets add some code.

    Open the main.lua file, and enter this text:

    local Mod = RegisterMod("Tutorial Mod", 1)
    print("The mod is working!")

    Finally, open the game, and start a run.

    If you open the console (Click here to learn more) there should be "The mod is working!" written.

  • Congrats! now you can start adding things
