Making a new Character

  • Step 1: players.xml

    Navigate to the /content/ folder in your mod.

    (if you don't have one, make sure you followed This)

    create a folder called "gfx", and a text document called "players.xml" (make sure you have file name extensions enabled. Windows should warn you about changing the file type)

    Open the players.xml file in your text editor of choice.

    Within the file, add:

    <players root="gfx/characters/costumes/" portraitroot="gfx/ui/stage/" nameimageroot="gfx/ui/boss/"> <player name="Bluesaac" skin="Character_-01_Isaac.png" hp="6" bombs="1" nameimage="PlayerName_-01_Isaac.png" portrait="PlayerPortrait_-01_Isaac.png" skinColor="-1" birthright="???" /> <player name="Bluesaac" skin="Character_-01b_Isaac.png" hp="6" bombs="1" nameimage="PlayerName_-01b_Isaac.png" portrait="PlayerPortrait_-01b_Isaac.png" skinColor="-1" birthright="???" bSkinParent="Bluesaac"/> </players>
    If you want to, change the name and spritesheet paths, but if you just want an idea of the process, I would recommend leaving them alone.
